Apa Itu ITC?

ITC merupakan Komunitas yang dibuat dijurusan SIJA SMKN 26 dimana dalam komunitas ini kami melakukan sharing,menambah ilmu dengan seluruh kelas Jurusan SIJA. Tak hanya itu disini kami pun saling belajar hal hal yang berhubungan dengan ilmu teknologi, terdapat 4 bidang yang dipelajari yaitu Software Application,Web Design and Development,Graphic Design,dan Networking.Dimana setiap bidang memiliki mentor nya masing-masing

Bidang ITC

Software Development, mengajarkan bahasa pemrograman kotlin dan juga pengembangan aplikasi android menggunakan Android Studio.

Web Development, mengajarkan pengembangan web seperti html, css, dan javascript menggunakan Visual Studio Code.

Design Graphic, mengajarkan design UI/UX, Vektor, lainnya menggunakan software Photoshop dan Adobe Illustrator atau Figma.

Networking, mengajarkan pembelajaran jaringan melalui simulasi Cisco dan virtual Linux server dengan Debian.
